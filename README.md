Мобильное приложение "кировский" предназначено для онлайн заказов в магазине кировский, а также для отслеживания скидок и акций.
В данном приложении можно видеть доступные товары, добавлять их в избранное, делать покупки. Получать уведомления о проходящих акциях и скадках в магазине.
Каждый пользователь может создать собственный аккаунт.
Инструкция к применению:
- установить приложение на мобильное устройство
- зайти в приложение
- автоматически открывается Главная страница, на ней можно перейти на страницу контактов магазинов и на вкладку с активными специальными предложениями. Баннер главной страницы предлагает самые выгодные предлозжения.
  Контакты:
   предоставляется список всех магазинов сети, их адрес, время работы, телефон для связи. А также общие данные для связи с магазином - главный сайт и почта.
- по нижнему меню навигации можно перейти на вкладки "Каталог", "Поиск", "Корзина" и "Профиль"
Каталог:
  на данной вкладке находиятся все товары магазина. Указано название, цена, раздел продукта и кнопка "Добавить в корзину".
  кликнув на понравившийся товар, можно перейти на карточку товара. К вышеописанной информации добавляется описание товара, его фотография и кнопка "Добавить в избранное".
Поиск:
  предоставляет возможно поиска определенных товаров.
Корзина:
  здесь будут находиться товары на выкуп.
Профиль:
  профиль пользователя предлагает кнопку входа в аккаунт, после входа будет указываться почта пользователя.
  через профиль можно попасть на страницы "Избранное" и "Контакты"
  Избранное:
  список товаров, лайкнутых пользователем
